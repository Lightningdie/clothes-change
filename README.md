# 小猫猫的衣橱

一个在线穿搭搭配应用，帮助你轻松管理和搭配服饰。

## 🌐 在线演示

**Demo地址：** [https://clothes-change.vercel.app/](https://clothes-change.vercel.app/)

## ✨ 主要功能

### 📱 个人信息管理
- 个人资料设置和编辑
- 头像上传和展示
- 信息保存成功提示

### 👔 服饰管理
- 浏览不同类别的服饰（上装、下装、配饰、鞋类等）
- 上传自定义服饰图片
- 服饰分类水平滚动浏览

### 🎨 穿搭编辑
- 点击选择服饰进行搭配
- 已选服饰展示区域
- 可拖拽图片展示区
  - 支持拖拽调整位置
  - 支持调整图片大小
  - 支持旋转图片方向
- 保存穿搭搭配

### 📂 穿搭收藏
- 查看已保存的穿搭
- 管理收藏的搭配方案

### 📱 响应式设计
- 完美适配PC端和移动端
- 优化的移动端交互体验
- 流畅的滚动和触摸操作

## 🛠️ 技术栈

- React 19
- TypeScript
- Ant Design
- React Router
- Vercel 部署

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# 构建生产版本
npm run build
```
